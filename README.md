# 新年の目標メーカー (New Year Goal Maker)

ランダムに表示される漢字を組み合わせて、新年の目標（抱負）をブレインストーミングするためのWebアプリケーションです。
偶然の組み合わせから生まれるインスピレーションと、生成AIによる言語化サポートを組み合わせることで、ユニークで前向きな目標作りを補助します。

## 🚀 デモサイト
**こちらからすぐに利用できます:**
[https://ronkyyyyy.github.io/new-year-goal-maker/](https://ronkyyyyy.github.io/new-year-goal-maker/)

## ✨ 特徴

* **直感的な操作**
    * 表示された漢字の中から、気になったものをタップして「確保（ロック）」するだけのシンプル操作。
    * 「残リセット」ボタンで、選ばなかった漢字だけを入れ替え、納得のいく組み合わせを探せます。
* **豊富な漢字データセット**
    * 具体物、動作、感覚、時間、場所など、様々なカテゴリから約240文字を厳選。
* **カスタマイズ可能**
    * 作成したい目標の数（最大100個まで！）
    * 1つの目標に使う漢字の数
    * ボードに表示する漢字の数
    * これらを自由に設定してスタートできます。
* **🤖 AI相談プロンプト生成機能**
    * 完成した漢字の組み合わせをもとに、ChatGPTやGeminiなどのAIに具体的な目標案を出してもらうための「特製プロンプト」をワンクリックでコピーできます。
    * 抽象的な漢字の羅列を、AIが具体的な行動目標やユニークな宣言文に変換してくれます。

## 📖 使い方

1.  **設定を決める**
    * ホーム画面で「目標の数」「1つの目標の文字数」などを設定し、「始める」を押します。
2.  **漢字を選ぶ**
    * アニメーションと共に表示される漢字の中から、ピンときたものをタップします。タップした漢字はグレーアウトされ、リストに追加されます。
3.  **入れ替える**
    * 画面上の漢字にしっくりくるものがなければ、「残リセット」ボタンを押してください。未選択の漢字が新しいものに入れ替わります。
4.  **AIに相談する（推奨）**
    * 全ての目標が埋まると、一覧画面が表示されます。
    * 「🤖 AI相談プロンプトをコピー」ボタンを押し、コピーされたテキストを普段お使いのAIチャットツール（ChatGPTなど）に貼り付けてください。意外な解釈や具体的なアクションプランが提案されます。

## 📦 ローカルでの実行方法

このアプリは静的ファイルのみで構成されているため、特別なビルド手順は不要です。

1.  このリポジトリをクローンします。
    ```bash
    git clone [https://github.com/ronkyyyyy/new-year-goal-maker.git](https://github.com/ronkyyyyy/new-year-goal-maker.git)
    ```
2.  フォルダ内の `index.html` をブラウザで開くだけで動作します。
